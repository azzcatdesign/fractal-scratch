# fractal-scratch

Generic partial version of Design System built for enterprise application.

## Getting started

1. Install fractal CLI (`npm i -g @frctl/fractal`)
1. Run `npm install`
1. Run `gulp`
1. Open `localhost` link in terminal output

## Build static docs

1. `fractal build`
